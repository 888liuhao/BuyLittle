## 学习是有套路

- 写页面套路
  基本思路：！！！！！！！！！！

1. 先分析有几个部分，在把某个部分细分为几个小部分。
2. 编写基本结构（html）
3. 去控制结构的基本布局如果对齐方式，横着排还是竖着排。
4. 最后结构成型之后再去改样式，如字体大小，颜色内外边距。
5. 对象psd设计稿是否接近还原！
